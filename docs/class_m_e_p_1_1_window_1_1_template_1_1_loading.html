Detailed Description

Template of a custom Middle Europe Productions loading window.

Functionalities: The window is capable of execution of given N number of methods on a seperate thread, furthermore, it keeps the track of the exceptions and automatically changes its status to MEP::Window::BaseWindow::Status::Exit when execution of a thread is completed.
